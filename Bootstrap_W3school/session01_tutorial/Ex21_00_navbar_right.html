Thanh điều hướng là tiêu đề điều hướng được đặt ở đầu trang

With Bootstrap, a navigation bar (thanh điều hướng) can extend (có thể mở rộng) or collapse (thu gọn), 
depending on (tùy thuộc vào) the screen size (kích thước màn hình).
        A standard navigation bar(thanh điều hướng chuẩn) is created with (được tạo bằng) the  .navbar class, 
        followed (theo sau là) by a responsive (phản hồi) collapsing class (1 lớp thu gọn):
.navbar-expand-xxl|xl|lg|md|sm (stacks (xếp chồng) the navbar (thanh điều hướng) vertically (chiều dọc) on xxlarge, 
extra large, large, medium or small screens).    
        To add links (để thêm liên kết) inside (bên trong) the navbar, use  an

    ) with class="navbar-nav".

Basic Navbar Example

A navigation bar is a navigation header that is placed at the top of the page.

The navbar-expand-xxl|xl|lg|md|sm class determines when the navbar should stack vertically (on xxlarge, extra large, large, medium or small screens).
